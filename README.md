# Filecoin-Data-Economy-Hackathon

## How to work

- install

  ```Bash
  yarn
  ```

- frontend build

  ```bash
  yarn frontend:build
  ```

- frontend dev

  ```bash
  yarn frontend:dev
  ```

- smartcontract build

  ```bash
  yarn backend:build
  ```

- smartcontract deploy

  ```bash
  backend:deploy:filecoinCalibration
  ```
